# BlackBox Portfolio

Welcome to **BlackBox Portfolio**.
This project is **free and open-source**, built with modern technologies, smooth animations, and a clean, minimal aesthetic.

---

## üöÄ Features
- Modern, responsive design that works across mobile S, M, L, tablets, laptops, and desktops
- Smooth scrolling and section animations
- Custom cursor and interactive elements
- **Audio player** with mute button
- **Scroll-up button** for easy navigation
- Sections include: About, Contact, Projects, Experience, and more
- Integration with [Discord-PFP-Fetcher](https://github.com/e-lyas/Discord-PFP-Fetcher) for dynamically displaying your Discord profile picture
- Easy-to-use structure ‚Äî designed for anyone to fork, modify, and deploy

---

## üõ† Technologies Used
- **Frontend:** HTML, CSS, JavaScript, TypeScript, React, TailwindCSS
- **Backend / API:** Cloudflare Worker (instructions included in Discord-PFP-Fetcher project)
- **Other Tools:** Git/GitHub, Vite, Discord Bots

---

## ‚ö° How to Use
1. **Clone the repository:**
   ```bash
   git clone https://github.com/e-lyas/BlackBox-Portfolio.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run locally:**
   ```bash
   npm run dev
   ```
4. **Deploy:** Build and host on any static hosting service.

**Note:** The portfolio fetches your Discord profile picture using the [Discord-PFP-Fetcher Worker](https://github.com/e-lyas/Discord-PFP-Fetcher). Instructions for setting up the Worker are included in that repository.
